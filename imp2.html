img attributes and its use

the types are:src,alt,width,height,style

< img >is used to add images
src:tells the browser where to find the image by giving it link or file path
alt: alternate text if the image does not appear it gives the description of the image
width and height:controls how big you want your image to appear
style: if you want to add border margin etc

< a > attributes and its uses

this is used to create links .A link that can take you to another website ,file,email etc
its attributes are :
href:tells where the link will go it can be a url,file or loc
target:decides where to open the file there are two types _blank and _self
title:when you clink on the link it will give some desciption or more info about it
